A simple one shot modal

This uses JQuery to trigger the modal and adds the 'close' trigger to the container.

  • Use class 'modal-closer' in the HTML to display the 'close' in the top right of the modal, remove class if not needed!
  • The css handles the content for the 'close' and the fade animation.
  • The escape key also closes the modal.
  • You'll need to add prefixes for the animation, this demo only includes -webkit-
